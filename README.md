# Installation

1. Clone the repository

For HTTP

```sh
git clone https://github.com/Mitushi-23/accredian_frontend.git
```

Backend repo:

```sh
git clone https://github.com/Mitushi-23/accredian_backend.git
```

2. Install dependencies

```sh
 npm i
```

3. Run server

```sh
npm run start
```

4. Run frontend

```sh
npm run start
```

# Environment Variables

```sh
  host: "localhost",
  user: "",
  password: "",
  database: "",
```
